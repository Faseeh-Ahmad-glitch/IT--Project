# UPI Transaction Persona Analysis

## Project Overview
This project performs user persona segmentation and classification using UPI transaction data. It identifies distinct spending and balance patterns using K-Means clustering and predicts these personas using a Random Forest Classifier.

## Key Findings
- **Segmentation:** K-Means identified 3 distinct user personas based on 'Withdrawal' and 'Balance' features.
- **Data Distribution:**
  - Persona 0: Typical transaction behavior (Majority)
  - Persona 1: Moderate activity
  - Persona 2: High-value/High-balance outliers
- **Model Performance:** The Random Forest Classifier achieved **100% accuracy**.
